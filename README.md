# Simple Stopwatch

A simple stopwatch made in C with the `ncurses` library.

## Prerequisites

- `gcc`
- `make`
- `ncurses` library

## Installation

```sh
git clone https://github.com/stancualex/simple-stopwatch \
&& cd simple-stopwatch \
&& make build
```

## Usage

- press `p` to toggle the stopwatch
- press `r` to reset the stopwatch
- press `q` to quit the application
